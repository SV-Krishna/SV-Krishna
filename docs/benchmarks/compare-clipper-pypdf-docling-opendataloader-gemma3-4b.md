Same 3 Clipper questions, comparing hybrid-mode answers.

## According to the Clipper Duet manual, what does the depth display show?

pypdf:
```
The Echo Sounder measures the depth from the transducer to the seabed. If the craft has a keel it is often more convenient to show the depth from the keel to the seabed. Sources: [2], 1
```

docling:
```
The depth display shows the depth and boat speed.  Sources: [3], Page unknown
```

opendataloader:
```
The Echo Sounder measures the depth from the transducer to the seabed. If the craft has a keel it is often more convenient to show the depth from the keel to the seabed. Sources: [2], Page 1
```

## According to the Clipper Duet manual, how do you change the shallow alarm setting?

pypdf:
```
To set the minimum depth alarm, press TRIP and TOTAL simultaneously. The display will briefly show CON to confirm the operation and then return to the SET ENG display. Sources: [2] p. 1
```

docling:
```
Use TRIP to decrement the setting and TOTAL to increment the setting. When the required minimum depth setting has been selected press SPEED.  Sources: [1] clipper-duet.pdf, Section: SETTING THE MINIMUM DEPTH ALARM, Page: unknown
```

opendataloader:
```
To set the minimum depth alarm, press TRIP and TOTAL simultaneously. The display will briefly show CON to confirm the operation and then return to the SET ENG display. Sources: [2] Page 9
```
